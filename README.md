# 🚀 React Router Simple Application

A beginner-friendly React application demonstrating **client-side routing** using React Router. This project showcases how to navigate between multiple pages without reloading the browser, following the concept of a Single Page Application (SPA).

---

## 📌 Features

* ⚛️ Built with React
* 🔀 Client-side routing using React Router
* 🧭 Navigation between multiple pages (Home, About, Contact)
* ⚡ Fast navigation without page reload
* 🧱 Clean and simple component-based structure

---

## 🛠️ Tech Stack

* React.js
* React Router DOM
* JavaScript (ES6+)
* HTML5 & CSS3

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/guri5405/03simplerouterworking.git
```

2. Navigate to the project folder

```bash
cd routerdemo
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

---

## 🔍 How It Works

* The application uses **BrowserRouter** to enable routing.
* **Routes** define which component should render for a specific URL path.
* **Route** maps paths like `/about` or `/contact` to components.
* **Link** is used for navigation without reloading the page.
